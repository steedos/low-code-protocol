## 许可证 Service

### hasFeature

判断平台是否有审计日志功能。

```
hasFeature('platform-ee.audit-logs')
```

### getProduct

获取产品的License

```
getProduct('contract-ee');
```

返回
```json
{
  product: 'contract-ee',
  end_date: '2022-12-31',
  user_number: 10,
  features: {
     feature_a: {
       label: '功能A',
       limit: 100,
       extra_data: {}
     }
  }
}
```
